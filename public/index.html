Concorrentes acompanhados

Para descobrir o ID: abra um anúncio do concorrente, copie o link do vendedor ou o nome da loja e use o Scan/catálogo para identificar o vendedor da oferta. Quando o Mercado Livre expõe o seller, o ID aparece como Seller ID. Se a API bloquear a listagem direta de anúncios desse vendedor, acompanhe por produto/catálogo no Scan.
A API oficial permite consultar dados públicos do vendedor e anúncios públicos. Vendas detalhadas, faturamento real por período e pedidos de concorrentes não são expostos sem autorização OAuth da própria conta.

Telegram via BotFather

Mini tutorial
  1. Abra o Telegram e procure por @BotFather.
  2. Envie /newbot, escolha nome e usuário do bot.
  3. Copie o token gerado pelo BotFather e cole abaixo. Essa configuração vale apenas para o usuário logado.
  4. Abra o bot criado no Telegram e envie uma mensagem qualquer para ele. Pode ser /start.
  5. Clique em Detectar Chat ID abaixo e selecione/copiei o chat.id retornado. Não use o ID do próprio bot.
  6. Escolha quais tipos de alerta este usuário deve receber.
